Come fare il deployment da PHPStorm

  1. Aprire il menu Tools -> Deployment -> Configuration...
  2. Premere il pulsante con il simbolo + per aggiungere una nuova configurazione di deploy
  3. Dare un nome significativo alla configurazione creata
  4. Nella sezione Type selezionare SFTP
  5. Se non già presente, aggiungere una SSH Configuration, selezionando ...
  6. Nel campo Host inserire l'indirizzo di Winnie 10.4.55.123
  7. Nel campo Username inserire lo username dato all'inizio dell'anno
  8. Come Authentication Type scegliere Password
  9. Provare con Test Connection se è stato configurato correttamente
  10. Eventualmente salvare la password
  11. Premere Ok e tornare alla schermata precedente
  12. Impostare la Root Path del progetto, selezionando l'icona della cartella e andando nella propria cartella, che si trova all'indirizzo /home/username/public_html/sottocartella
  13. In WebServer impostare l'indirizzo del proprio progetto su Winnie, ad esempio http://10.4.55.123/~simvac/introduzione
  14. Spostarsi sul tab Mappings
  15. Localpath lasciare quello preimpostato
  16. Negli altri due campi, a meno di casi eccezionali, inserire / e /
  17. Premere infine Ok e si avrà una configurazione di deployment
